# Marketplace Comparison

## Positioning

Bankable Property Network should not compete with listing marketplaces as another place to browse condos.

The stronger position:

> Discovery is not trust. Listings help buyers find property. Bankable Property Network helps buyers safely move money into verified property under bank-controlled settlement conditions.

## Listing Marketplace Model

Examples: large property portals and agent-led discovery platforms.

Strengths:

- many listings;
- SEO and lead generation;
- fast discovery;
- agent pipeline;
- consumer-friendly browsing.

Weaknesses:

- listings do not guarantee a safe transaction;
- developer, seller, and agent verification may be shallow or inconsistent;
- buyer still faces deposit pressure alone;
- payment instructions and legal entity mismatches can appear after discovery;
- bank route, FET readiness, escrow, and evidence are outside the core product.

## Bankable Property Network Model

Strengths:

- verified property, developer, and agent layers;
- buyer capital classification;
- bank-led settlement routing;
- escrow and conditional release;
- Closing Passport evidence hash;
- post-purchase financial lifecycle.

Weaknesses:

- harder to explain than a listing site;
- needs banking anchors and phased rollout;
- must avoid claiming legal/title guarantees.

## Strategic Conclusion

Listing marketplaces own discovery. Bankable Property Network should own trusted settlement.

The network can sit beside or on top of listing platforms:

- buyer finds a property anywhere;
- Bankable Property Network verifies the transaction path;
- banking anchors approve the settlement route;
- Closing Passport proves the verified process.
